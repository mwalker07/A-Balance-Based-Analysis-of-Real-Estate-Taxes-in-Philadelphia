# A-Balance-Based-Analysis-of-Real-Estate-Taxes-in-Philadelphia

Soon after starting college, I realized that I am particularly interested in the data analytics aspect of computer science. I have taken an introductory data science course, though this is my first dashboard-style project. Which required me to both discover and learn platforms such as streamlit, pandas, matplotlib, etc, in order to make this task possible.

My goal was to create a meaningful data analysis project that is centered around the citizens of Philadelphia, like myself. Using a dataset from OpenDataPhilly and drawing inspiration from the work done in the City of Philadelphia's department of revenue. In this project I will be analyzing real estate accounts with tax balances, and after my conclusion, give some sort of predictive insight with hopes of prevention.
